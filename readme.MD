**Space Travel**

El proyecto tiene como objetivo incentivar la exploracion espacial atravez de video, imagenes y datos interesantes

Para el desarrollo de la web se recurrio a apuntes de clase, filminas, www.w3schools.com y otras fuentes de internet.

Las proximas actualizaciones seran: una resolucion adaptable,la opcion de convertir la unidad de tiempo a horas y dias, un calculo mas preciso ya que actualmente se esta utilizando solo formulas de MRU, inicializar el canvas al clickear launch.
